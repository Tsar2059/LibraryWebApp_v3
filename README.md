# 📚 Библиотека — веб-сайт для публичной библиотеки

Курсовой проект по дисциплине «Кроссплатформенная среда исполнения программного обеспечения»

## 🔧 Технологии

- **.NET 8** (Blazor Server)
- **Entity Framework Core** (SQLite)
- **Docker** (контейнеризация)
- **FluentValidation** (валидация)

## 📁 Функционал

### Для читателей:
- Просмотр каталога книг с поиском
- Чтение и скачивание книг (PDF, EPUB, FB2)
- Просмотр информации об авторах
- Обратная связь через email

### Для администратора:
- Аутентификация (логин/пароль)
- Управление книгами (добавление, удаление, редактирование количества)
- Управление авторами (добавление, удаление)
- Управление категориями

## 🔐 Учётные данные администратора

| Поле | Значение |
|------|----------|
| **Логин** | `admin` |
| **Пароль** | `admin123` |

> После входа перейдите на страницу `/admin` для доступа к панели управления.
